Dashboard Pendanaan/Kredit UMKM

Silakan pilih menu di bawah ini untuk memulai. Masing-masing menu memiliki sumber data berbeda. Dashboard 1 menggunakan data Susenas dengan proxy UMKM variabel Berusaha untuk memberikan gambaran akses kredit dan jenisnya secara umum. Dashboard 2 menggunakan data SSKI Bank Indonesia untuk memberikan gambaran nilai dan kondisi kredit yang diberikan.
